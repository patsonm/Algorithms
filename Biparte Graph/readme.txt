This zipped folder contains the files for Homework 5 or Oregon State online Bacc Program

The program compiles an input list of babyfaces and heels (wresting terms) and evaluates if a biparte graph can be formed.  If the graph can be formed, resulting teams are displayed

To complile file c++ file

make wrestling

to run file

the program is to be compiled with a text file containing names of rivalries and wrestler names

./wrestling (input text file)
